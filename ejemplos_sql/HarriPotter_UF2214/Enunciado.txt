UF2214 - Practico - Implementación y uso de una BD Cuestionario

La base de datos ya esta implementada y es necesario importarla en nuestro servidor. Para lo cual nos han enviado un script ( vease ANEXO UF2214 )

Importar base de datos a nuestro servidor                                                           -  1,5 punto

realizar las siguientes consultas SQL ( entregar fichero texto )
Listado de todas las casas de Howards                                                       - 1 punto
Tres primeros Alumnos con mejor nota en el año 2017                               - 1 punto
Listado de profesores y materias impartidas ordenado por materias alfabéticamente para un año concreto - 1 punto
